# Competitor Documentation Pack

This folder is the local front door for studying the competitor stack that
matters most for Dagua:

- Graphviz `dot`
- Graphviz `sfdp`
- ELK Layered
- dagre
- NetworkX `spring_layout`

These files do not try to mirror the upstream documentation. Instead, they
collect the parts that matter most for Dagua iteration:

- official documentation links
- source-code entrypoints worth reading
- the specific geometry/layout topics to study
- local Dagua-generated outputs worth comparing against

Recommended reading order:

1. `graphviz.md`
2. `elk.md`
3. `dagre.md`
4. `networkx.md`
5. `../COMPETITOR_GEOMETRY_MEMO.md`
