# Dagua Video Resources

This folder is the bridge between the repo-native explainer and any future YouTube material.

The idea is:
- `docs/how_dagua_works.md` is the canonical public written explainer
- this folder holds reusable video resources derived from the same ideas

Contents:
- `script.md` — narration draft
- `shotlist.md` — scene plan and visuals

This keeps the video effort grounded in assets that also help the docs, README, and social demos.
